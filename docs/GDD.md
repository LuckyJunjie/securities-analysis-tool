## 1. 项目概述

### 1.1 项目愿景

构建一个基于**价值投资理念**的量化证券分析系统，整合三大投资哲学：

- **本杰明·格雷厄姆** - 安全边际、价值投资
- **菲利普·费雪** - 成长股洞察、管理层评估
- **肖星** - 财报三张表分析框架

### 1.2 核心价值

- **可量化** - 将投资理念转化为具体财务指标
- **可验证** - 基于历史数据验证投资逻辑
- **预警未来** - 通过"信号机制"预警风险与机会

## 2. 功能需求

### 2.1 财务数据分析模块

#### 2.1.1 十年财务指标库

| 指标 | 说明 | 数据来源 |
|------|------|----------|
| 营业收入 | 年度/季度营收及同比增速 | 财报 |
| 归母净利润 | 净利润及同比增速 | 财报 |
| 毛利率 | 毛利/营收 | 财报 |
| 净利率 | 净利润/营收 | 财报 |
| ROE | 净资产收益率 | 财报 |
| 存货周转率 | 运营效率指标 | 财报 |
| 资产负债率 | 财务健康度 | 财报 |
| 流动比率 | 短期偿债能力 | 财报 |

#### 2.1.2 趋势分析引擎

- [ ] 自动计算过去10年CAGR
- [ ] 识别连续季度负增长信号
- [ ] 毛利率持续下滑预警
- [ ] 存货周转异常检测

### 2.2 估值分析模块

#### 2.2.1 估值指标

| 指标 | 说明 |
|------|------|
| PE (TTM) | 市盈率 |
| PB | 市净率 |
| DCF | 现金流折现估值 |
| 安全边际 | (内在价值 - 当前价格) / 内在价值 |

#### 2.2.2 估值预警

- [ ] 期望增长率 vs 实际增长率偏离检测
- [ ] 戴维斯双杀预警
- [ ] 格雷厄姆评级 (A/B/C)

### 2.3 宏观数据模块

#### 2.3.1 宏观指标监控

| 指标 | 频率 | 用途 |
|------|------|------|
| 发电量 | 月度 | 经济同步指标 |
| PPI | 月度 | 生产者价格指数 |
| CPI | 月度 | 消费者价格指数 |
| 利率政策 | 实时 | 货币政策 |
| 消费信心指数 | 月度 | 消费趋势 |

#### 2.3.2 宏观预警

- [ ] PPI连续负增长预警 (通缩信号)
- [ ] CPI转负预警
- [ ] 消费板块宏观承压预警

### 2.4 预警信号系统

#### 2.4.1 信号类型

| 级别 | 名称 | 触发条件 |
|------|------|----------|
| 🔴 红色 | 强烈卖出 | 业绩双降 + 宏观不利 |
| 🟡 黄色 | 关注 | 增速放缓 + 估值偏高 |
| 🟢 绿色 | 买入 | 安全边际 > 30% + 财务健康 |

#### 2.4.2 信号组合

- **成长股逻辑破坏**: 连续2季度营收/利润负增长
- **预期差**: 市场预期增长 > 实际增长 > 10%
- **戴维斯双杀**: 业绩下调 + 估值下杀

### 2.5 持仓管理模块

#### 2.5.1 持股数据库

```
字段:
- 股票代码
- 股票名称
- 持仓日期
- 持仓价格
- 当前价格
- PE (TTM)
- 近3年EPS CAGR
- 预期增长率 vs 实际
- 毛利率(TTM)
- 营收同比(Q)
- 净利率同比(Q)
- 预警信号
- 宏观背景
- 格雷厄姆评级
- 行动指引
```

#### 2.5.2 功能

- [ ] 批量导入持仓
- [ ] 季度自动更新财务数据
- [ ] 一键生成持仓分析报告

## 6. 验收标准

### 6.1 功能验收

- [ ] 能获取任意A股/港股10年财务数据
- [ ] 能计算安全边际并给出评级
- [ ] 能识别"业绩双降"预警信号
- [ ] 能监控宏观指标并预警
- [ ] 能管理持仓并生成分析报告

### 6.2 性能验收

- [ ] 单股票分析 < 5秒
- [ ] 批量持仓分析 < 30秒
- [ ] 宏观数据更新 < 10秒

### 6.3 示例：中国中免分析

基于本系统，应能在2026年1月发出以下预警：

| 信号 | 内容 |
|------|------|
| 业绩双降 | 2024年净利润同比-36.44%，2025前三季度-22.13% |
| 宏观承压 | PPI连续30+月负增长，消费信心低迷 |
| 预期差 | 期望增长率22.4% vs 实际-25% |
| 格雷厄姆评级 | C (规避) |
| 行动 | 减仓/观望 |
