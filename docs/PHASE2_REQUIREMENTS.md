| 需求ID | 描述 | 优先级 |

## 3. 技术设计

### 3.1 缓存策略
```
Redis (推荐):
  key: "quote:{code}"
  value: {price, timestamp}
  ttl: 300秒 (5分钟)

本地内存 (备选):
  dict = {code: {price, timestamp}}
```

### 3.2 数据流
```
用户请求 → 检查缓存 → 有→返回
                ↓无
          调用外部API → 存入缓存 → 返回
```
