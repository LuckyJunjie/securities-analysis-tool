## 1. 系统概述

基于本杰明·格雷厄姆价值投资理念和《一本书读懂财报》分析方法的辅助决策平台。

### 核心价值
- 自动化采集十年财报数据
- 宏观指标监控 (发电量/PPI/CPI/利率)
- 智能预警信号
- Web应用/API服务

## 5. 核心服务设计

### 5.1 指标计算服务
- **输入**: 原始财务数据
- **输出**: 毛利率、净利率、ROE、周转率等
- **触发**: 定时任务 (每日) 或 按需

### 5.2 趋势识别服务
- **输入**: 指标历史序列
- **输出**: 趋势标签 (上升/下降/平稳)、拐点
- **算法**: 线性回归、移动平均、规则判断

### 5.3 估值模型服务
- **模型**: 格雷厄姆公式、反DCF
- **输入**: 股价、EPS、增长率、无风险利率
- **输出**: 内在价值、安全边际、评级

### 5.4 预警引擎
- **规则库**:
  - 财务: 营收连2季负增长、毛利率连3季下滑
  - 估值: PE > 历史80%分位 + 增长乏力
  - 宏观: PPI连6月负、CPI < 1%

## 6. 业务流程 (中国中免案例)

```
1. 数据采集 → 财报PDF解析 → 原始数据存储
2. 指标计算 → 计算营收/利润/毛利率 → 同比增速
3. 趋势识别 → 检测连2季营收下降
4. 估值对比 → PE vs 预期增长率
5. 预警触发 → 生成预警记录
6. 推送展示 → WebSocket + 邮件通知
```

## 7. API 设计

### 7.1 用户服务
- `POST /api/auth/register` - 注册
- `POST /api/auth/login` - 登录
- `GET /api/user/profile` - 用户信息

### 7.2 股票服务
- `GET /api/stocks` - 股票列表
- `GET /api/stocks/{code}` - 股票详情
- `GET /api/stocks/{code}/financials` - 财务数据

### 7.3 分析服务
- `GET /api/analysis/{code}` - 综合分析报告
- `GET /api/valuation/{code}` - 估值分析
- `GET /api/trend/{code}` - 趋势分析

### 7.4 预警服务
- `GET /api/alerts` - 预警列表
- `POST /api/alerts/rules` - 创建规则
- `PUT /api/alerts/{id}/read` - 标记已读
